Aku langsung menganalisisnya di aplikasi Ghidra. Aku mulai mencari di bagian code mana yang terdapat printable character "Enter password: " namun, aku tidak dapat menemukan printable character tersebut dan malah menemukan sebuah codebase yang sangat besar yang kemungkinan berisi sebuah library di Python.

Aku mulai menjalankan command strings chall | grep -i pyinstaller untuk mengetahui apakah file tersebut adalah PyInstaller dan ternyata benar saja.

Kemudian aku mulai mencari write up di search engine dengan keyword "pyinstaller reverse engineering ctf" dan mulai mengamati bagaimana cara menghandle file PyInstaller.
